Проект домашнего задания loftblog 2015

Для загрузки и использования проекта необходимо:

0. Установить git

1.	Загрузить репозиторий:
git clone https://github.com/Dajjedra/DZ_01

2. Установить Node.JS + npm:
https://nodejs.org/download/

3. Установить bower:
npm install -g bower

4. Установить gulp:
npm install -g gulp
